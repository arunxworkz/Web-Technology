Youtube

YouTube is an American online video sharing platform owned by Google. Accessible worldwide,[note 1] YouTube was launched on February 14, 2005, by Steve Chen, Chad Hurley, and Jawed Karim, three former employees of PayPal. Headquartered in San Bruno, California, United States, it is the second-most visited website in the world, after Google Search.As of January 2024, YouTube has more than 2.7 billion monthly users, who collectively watch more than one billion hours of videos every day.[7] As of May 2019, videos were being uploaded to the platform at a rate of more than 500 hours of content per minute,[8][9] and as of 2021, there were approximately 14 billion videos in total.[9]

Operation of JVM

  1. Loading

    The JVM starts by loading the .class files, which are the compiled Java bytecode.
    These class files are loaded by the ClassLoader, which is responsible for loading classes into the JVM at runtime.
    ClassLoaders can load classes from various sources (e.g., files, network, or other locations)..

  2. Bytecode Verification

    JVM will perform the byte code verification to check whether the bytecode is valis or not

  3. Execution

    JVM consistes of two primary appproaches for executing the byte code
    Interpreter - It will translates the byte code line by line into machine code instruction
    JIT - Let us conside an exapmle that a byte code is having an function which is invoked every now and then and here JIT will compile int into machine code and cached will execute it directly.This speeds up performance since the JVM does not have to interpret the same bytecode repeatedly.
